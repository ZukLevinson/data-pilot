# Portal (Frontend)

The **Portal** is the user-facing application for the Data Pilot project.

## 🚀 Features
- **Interactive Maps**: Built with Leaflet to visualize spatial data.
- **AI Chatbot**: Integrated chat interface for querying data.
- **Responsive Design**: Styled with PrimeNG.

## 🛠️ Development
Run `npx nx serve portal` for a dev server. Navigate to `http://localhost:4200/`.

## 📦 Build
Run `npx nx build portal` to build the project. The build artifacts will be stored in the `dist/apps/portal` directory.
